# SGAFO - Application Logic

This directory contains the core application code for **SGAFO** (Système de Gestion des Activités des Formateurs OFPPT).

## 📖 Global Documentation

For full project documentation, technical stack details, and installation instructions, please refer to the **[Main README](../README.md)** at the root of this repository.

## 💻 Development Notes

### Backend
- Built with **Laravel 12**.
- Uses **Inertia.js** for seamless integration with the React frontend.
- Key modules are located in `app/Http/Controllers/Modules`.

### Frontend
- Built with **React 18** and **TypeScript**.
- Components and Pages are located in `resources/js`.
- Uses **Tailwind CSS** for styling.

### Commands
- `composer run dev`: Runs the development server (Laravel + Vite).
- `npm run build`: Builds the production assets.
- `php artisan migrate`: Runs database migrations.
